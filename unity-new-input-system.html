Unity's new Input System

Learn to setup and use Unity's new Input System (tested on Unity 6000.4.6f1)

First, run Unity.

  1. If you want to edit Input Action Asset, double click on it in the Project window. Editing in Project Settings window is lacking the Save option!
  2. Remember that you can have multiple Input Action Assets in the project.
  3. Use [SerializeField] private InputActionReference _nameOfYourAction; in your script. You will be able to drag and drop an action from Project window to your Inspector. Alternatively, you can press circle near your field in the inspector to see all available actions.
  4. You can read value of your Value - Vector2 action with _nameOfYourAction.action.ReadValue(); in Update method.
  5. You can read value of your Button action with _nameOfYourAction.action.WasPressedThisFrame(); in Update method. IsPressed() and WasReleasedThisFrame() can also be useful.
